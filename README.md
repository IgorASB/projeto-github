
# Calculadora de Média Aritmética

Um programa simples em C que calcula a média aritmética de três números.

## Descrição

Este programa solicita ao usuário que digite três números e calcula sua média aritmética, exibindo o resultado com duas casas decimais.

## Como Usar

1. Compile o programa:
    ```bash
    gcc programa.c -o media
    ```

2. Execute:
    ```bash
    ./media
    ```

3. Digite os três números quando solicitado.

## Exemplo de Saída

```
Digite o primeiro número: 7
Digite o segundo número: 8
Digite o terceiro número: 9
A média aritmética é: 8.00
```

## Requisitos

- Compilador C (gcc, clang, etc.)
